# Data Products Overview

This section describes the data products available through CAMPFIRE.

## Product Types

*Documentation coming soon!*

CAMPFIRE provides several types of data products:

### Spectroscopic Products

- **1D Spectra** - Extracted, calibrated spectra
- **2D Spectra** - Rectified spectral images
- **Redshift Fits** - Template fitting results

### Imaging Products

- **Mosaics** - Combined NIRCam images
- **Thumbnails** - Cutout images centered on sources
- **RGB Images** - Color composite images
