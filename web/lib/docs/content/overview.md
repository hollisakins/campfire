# CAMPFIRE Documentation

Welcome to the CAMPFIRE documentation. CAMPFIRE (COSMOS Archive of MultiPle-Field Internal Reductions & Extractions) is a platform for accessing and inspecting reduced JWST spectroscopic data.

## Quick Links

- [Getting Started](/docs/getting-started) - Set up your account and access data
- [Visual Inspection Guide](/docs/inspection) - Learn how to inspect and classify spectra
- [Data Reduction](/docs/reduction) - Understand how the data is processed
- [Data Products](/docs/data-products) - Reference for output files and formats
