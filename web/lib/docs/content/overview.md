# CAMPFIRE Documentation

Welcome to the CAMPFIRE documentation. CAMPFIRE (the COSMOS Archive of MultiPle-Field Internal Reductions & Extractions) is a platform for accessing and inspecting reduced JWST imaging & spectroscopic data, primarily in the COSMOS field but inclusive of other fields as well. 
The CAMPFIRE portal and data products are developed and maintained by [Hollis Akins](mailto:hollis.akins@gmail.com).

## What/who is CAMPFIRE for?

CAMPFIRE was designed primarily to provide the COSMOS team access to spectra from several cycle 4 programs (EMBER, ZENITH) obtaining large spectroscopic samples. 
Other public datasets (CAPERS, RUBIES, etc) are included as well to provide a uniformly-reduced and inspected sample of objects with JWST spectroscopy. 

Some NIRCam imaging data is provided through CAMPFIRE as well, though a proper integration between images > photometric catalogs > spectroscpic catalogs is still to come. 

> **Usage:** If you're interested in using CAMPFIRE data products for your work, check out the [Usage Guidelines](/docs/usage). 

## Key Features

- Uniform reduction of >15,000 spectra with automatic redshift fits
- Advanced search & filtering options (including coordinate search) to query the spectroscopic database
- Bulk download options (python API coming soon!)
- Interactive spectrum plots and inspection tools
- Collaborative object flagging, redshift grading, and comments all going to a centralized database
