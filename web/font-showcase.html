Quick Comparison

  • Current (System fonts) - No external font loading, but appearance varies by OS. SF Mono can appear thin on macOS.
  • Roboto family - Very neutral and clean. Roboto Mono is slightly condensed. Excellent for data-heavy interfaces. ~50KB total.
  • Source family - Slightly more character and warmth. Source Code Pro has excellent number differentiation. Popular in scientific/technical contexts. ~60KB total.

Both Google Font options will load quickly and provide consistent appearance across all platforms.
